# LLM Banking POC

A proof-of-concept banking application showcasing the integration of a Large Language Model (LLM) and AI-driven features to enhance the user experience. This project demonstrates how AI can be leveraged to provide personalized financial advice, predict spending patterns, and offer tailored product recommendations.

## Overview

The LLM Banking POC is a frontend application built with React and styled using Tailwind CSS. It showcases a modern banking interface with interactive components for user engagement. The key features of this POC include:

- **Spending Prediction**: Utilizes a regression model to predict the user's spending for the next month based on their historical spending data. The prediction is visualized through an interactive chart.

- **AI-Generated Financial Advice**: Integrates with an LLM (GPT-4-turbo) to provide personalized financial advice based on the user's spending habits and predicted spending. The LLM generates human-friendly advice to help users make informed financial decisions.

- **Featured Product Carousel**: Displays a carousel of featured banking products with AI-generated recommendation texts. The LLM is used to create persuasive and tailored product descriptions based on the user's profile and financial goals.

- **Interactive Chat Widget**: Allows users to engage in conversational interactions with the AI assistant. Users can ask questions related to their spending, saving goals, or general financial queries, and receive dynamic responses generated by the LLM.

## Purpose

The primary purpose of this POC is to demonstrate the potential of integrating AI and LLMs in banking applications. By leveraging the power of language models, we can provide users with intelligent and personalized financial guidance, enhance their understanding of their spending patterns, and offer relevant product recommendations.

The LLM-driven features aim to improve the user experience by offering valuable insights, engaging interactions, and data-driven recommendations. This POC serves as a starting point for exploring the possibilities of AI in the banking domain and can be further expanded and refined based on user feedback and business requirements.

## Technologies Used

- **Frontend**: React, Tailwind CSS
- **Backend**: Node.js, Express
- **AI and LLM**: GPT-4-turbo (OpenAI)
- **Regression Model**: ml-regression library
- **Data Visualization**: Chart.js
